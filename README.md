# resumo-do-lab
Este repositório contém o resumo das lições aprendidas durante o desenvolvimento do lab na DIO

Neste lab eu aprendi à criar um repositório no github, clonar o projeto para uma pasta local, adicionar um conteúdo, adicionar à area de stage, criar um commit para gravação do conteúdo no git e enviar as alterações para o repositório remoto.
Além disso, aprendi sobre projetos open source, formas de contribuir em projetos open source com a criação de issues e pull request, criação de fork de um repositório e sobre conveções de commits.
